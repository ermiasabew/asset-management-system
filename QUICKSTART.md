# Quick Start Guide

## Get Started in 3 Steps

### Step 1: Install Dependencies
```cmd
npm install
```

### Step 2: Initialize Database
```cmd
npm run init-db
```

### Step 3: Start Server
```cmd
npm run dev
```

### Step 4: Open Browser
Navigate to: **http://localhost:3000**

Login with:
- **Username:** admin
- **Password:** admin123

---

## Quick Feature Guide

### Adding an Asset
1. Click "Assets" in sidebar
2. Click "Add Asset" button
3. Fill in asset details
4. Save

### Adding Inventory Item
1. Click "Inventory" in sidebar
2. Click "Add Item" button
3. Enter item details
4. Save

### Adding Employee
1. Click "Employees" in sidebar
2. Click "Add Employee" button
3. Fill employee information
4. Save
5. Add guarantor documents if needed

### Adding Client
1. Click "Clients" in sidebar
2. Click "Add Client" button
3. Enter client details
4. Save
5. Create service contracts
6. Assign employees

### Managing Rentals
1. Click "Rentals" in sidebar
2. Add rental properties
3. Add tenants to properties
4. Track rent payments

### Viewing Reports
1. Click "Reports" in sidebar
2. View dashboard statistics
3. Export reports as needed

## Tips

- Use the search and filter options to find items quickly
- Check notifications bell for alerts
- Low stock items are highlighted in yellow
- Document expiry reminders appear in notifications
- All changes are logged in audit trail
